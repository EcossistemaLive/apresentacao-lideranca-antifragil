# Liderança Antifrágil - ViDi

Apresentação interativa sobre Liderança Antifrágil, desenvolvida para a **ViDi Academy**.

## Visual
- Estética **Modern Classic** com Glassmorphism.
- Paleta de cores premium (Gold & Black).
- Navegação fluida com Scroll Snap.

## Tecnologias
- HTML5 / CSS3 (Vanilla)
- Intersection Observer API for animations
- Responsive Design (Mobile First)
